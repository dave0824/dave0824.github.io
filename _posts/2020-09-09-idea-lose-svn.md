## 前言
今天好好的idea突然失去了项目的svn控制，记下解决方法




## 解决
打开 File -> Settings -> Version Control
在这里就可以看到被 svn 管控的项目

![svn](https://github.com/dave0824/dave0824.github.io/blob/master/asset/idea/svn.png?raw=true)

点击上图的加号，选择 Directory ,找到要添加到svn管控的项目，加入即可。
